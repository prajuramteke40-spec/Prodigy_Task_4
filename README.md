# Prodigy Infotech: Machine Learning - Task 04

## ✋ Hand Gesture Recognition Model

This repository contains the implementation of Task 04 for the Prodigy Infotech Machine Learning Internship. The objective of this project is to develop a machine learning model that can accurately identify and classify different hand gestures from image data, enabling intuitive human-computer interaction.

### 📋 Project Overview
Image classification is a complex but highly rewarding area of machine learning. In this project, we process image data, extract meaningful features, and train a classification model to recognize specific hand signs. 

This notebook walks through the entire pipeline: loading the image dataset, preprocessing and resizing the images, building the classification model, and evaluating its performance on unseen test data.

### 🛠️ Technologies & Libraries Used
* **Python 3.x:** Core programming language.
* **NumPy & Pandas:** For data manipulation and numerical array operations.
* **OpenCV / PIL:** For image processing, loading, and formatting.
* **Scikit-Learn / TensorFlow:** For building, training, and evaluating the machine learning model.
* **Matplotlib & Seaborn:** For visualizing the images, model predictions, and confusion matrices.
